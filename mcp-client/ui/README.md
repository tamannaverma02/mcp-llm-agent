## Streamlit UI for Client

A minimal UI to interface with the MCP Client via Streamlit.

* * *

### Setup

1. 1.  **Install Streamlit & dependencies**
1.     

`uv add streamlit`

1. 2.  **Run the UI**
1.     

`streamlit run main.py`
